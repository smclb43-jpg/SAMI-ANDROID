AFTEM Mobil İstasyon Android Projesi

Bu klasör gerçek APK üretmek için Android Studio projesidir.

İçerik:
- Native Android WebView uygulaması
- Telefon açılışında istasyon giriş ekranı
- Sadece ilgili istasyon listesini gösteren sade mobil arayüz
- PC'de çalışan AFTEM mobil sunucusuna bağlanma desteği

Kullanım:
1. Android Studio ile bu klasörü aç
2. Gradle sync tamamlanınca Run veya Build APK yap
3. Telefonda ilk açılışta istasyon ve API adresini gir
4. API olarak PC'deki sunucu adresini yaz: http://PC_IP:8765

Not:
- Bu oturumda APK derlenmedi
- Derleme için Android Studio veya Gradle Android ortamı gerekir
